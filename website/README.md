## Local Development

1. Clone the repository:
```bash
git clone https://github.com/jape-ai/jape-ai
cd website
```

2. Start a local server:
```bash
# Using Python
python -m http.server 8000

# Or using any other local server
```

3. Visit `http://localhost:8000` in your browser
